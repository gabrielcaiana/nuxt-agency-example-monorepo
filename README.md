# Sample monorepo with Nuxt3

- this repository contains a simple monorepo with turborepo, using two apps with nuxt 3 and a shared component lib.


# Commands

install dependencies:
```
yarn install
```

build projects:
```
yarn build
```

run all apps
```
yarn dev
```
run all tests:
```
yarn test
```

run ui tests
```
yarn test:ui
```

generate types
```
yarn prepare
```
